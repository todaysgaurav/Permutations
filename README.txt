BUILD DESCRIPTION
=================

1. Download Bobble.java and input.csv file.
2. Both file are in same location.
3. On commend prompt or terminal compiled the java file using 
	javac Bobble.java
4. Run this file using 
	java Bobble input.csv

Note: 
*In your system must be installed java.
*Give any input file but make sure input file in csv formate.
